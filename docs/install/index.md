# Install

## Dependencies
### List
- git
- python
- pip
- pip3

### Install Dependencies
    $sudo apt-get install python-pip python3-pip

## Install vim-pack

    $ git clone https://github.com/tanvir002700/vim-pack.git
    $ ./vim-pack/install.sh
